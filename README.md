# dentist_web_app

Taşkın Özcan için modern bir diş hekimliği web sitesi tasarlandı.

## İçerik

- `index.html`: Ana sayfa, hero bölümü, hizmetler, randevu formu ve iletişim alanı.
- `styles.css`: Modern, mobil uyumlu stil düzenlemeleri.
- `script.js`: gezinme, form işlemleri ve aktif bölüm vurgulama.
- `assets/`: avatar ve diş dekoratif görselleri için resim dosyaları.
- `hakkimizda.html`, `hizmetler.html`, `iletisim.html`, `randevu.html`: ilave sayfalar için örnek yapılar.

## Kullanım

1. Proje dizinine gidin.
2. `python3 -m http.server 8000` komutuyla yerel sunucu başlatın.
3. `http://127.0.0.1:8000` adresinden siteyi görüntüleyin.
